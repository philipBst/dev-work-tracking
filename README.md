# Steps to configure application

## Go to backend directory

```bash
cd backend
```

## Install dependencies

```bash
pnpm install
```

## Start Application

```bash
pnpm run start
```

## Once backend is running open new terminal and go to root of project

```bash
cd ..
```

## Go to client directory

```bash
cd client
```

## Install dependencies

```bash
pnpm install
```

## Start Application

```bash
pnpm run dev
```
